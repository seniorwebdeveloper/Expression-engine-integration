Module Features

The pMailer Contacts Export Form module allows you to export all of your existing ExpressionEngine members directly into your pMailer mailing list(s).

 

Installation

  1. Copy the pmailer_contacts folder to your ExpressionEngine [system]/expressionengine/third_party folder
  2. Install the module from your ExpressionEngine control panel
  3. Enter your API Url and Key in the Api Settings
  4. Select the mailing list(s) you want to export your users to, select whether you want to enable a Double Opt-In and save
  5. Click the Export Contacts button to start exporting.
