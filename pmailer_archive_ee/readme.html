Module Features

The pMailer Message Archive module allows you to display all of your pMailer newsletters directly on an ExpressionEngine page.

 

Installation

  1. Copy the pmailer_archive folder to your ExpressionEngine [system]/expressionengine/third_party folder
  2. Copy the pmailer_archive_root folder to the root of your ExpressionEngine site
  3. Install the module from your ExpressionEngine control panel
