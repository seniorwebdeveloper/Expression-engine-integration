Module Features

The pMailer Subscription Form module allows you to place a form on your site that subscribes users directly into your pMailer mailing lists(s).

 

Installation

  1. Copy the pmailer_subscribe folder to your ExpressionEngine [system]/expressionengine/third_party folder
  2. Install the module from your ExpressionEngine control panel
  3. Enter your API Url and Key in the Api Settings
